PriceWise – E-Commerce Price Tracking System
Overview
PriceWise is a backend-focused e-commerce price tracking system that monitors product prices
from online stores and notifies users when prices drop or products come back in stock. The main
focus of this project is backend engineering and logic development. While frontend interfaces can
be easily replaced or redesigned, the core complexity lies in building strong backend systems such
as scraping pipelines, data processing, automation, and notification services.
Core Idea
Online shopping prices change frequently and users often miss good deals. PriceWise solves this
problem by automatically tracking product prices, storing historical price data, detecting price drops,
and notifying users through email alerts.
Key Features
• Product price scraping from e-commerce URLs
• Storage of product data and historical price records
• Automated cron jobs to periodically update product prices
• Detection of price drops and stock availability
• Smart email notifications to users
• Backend-first architecture focusing on system logic
Tech Stack
• Frontend: React, Tailwind CSS
• Backend: Node.js, Express.js
• Database: MongoDB, Mongoose
• Web Scraping: Axios, Cheerio
• Automation: Cron Jobs
• Notifications: Nodemailer
System Architecture
• User submits a product URL
• Backend scraper extracts product information
• Database stores product details and price history
• Cron jobs periodically re-scrape products
• Price analysis detects price changes
• Notification system sends email alerts to users This project was mainly focused on backend system design and logic development. It helped
strengthen skills in web scraping, backend architecture, automation pipelines, database schema
design, and building notification systems. The emphasis was placed on building a reliable backend
because frontend interfaces can easily be replaced or redesigned.
